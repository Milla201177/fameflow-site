# FameFlow Site

This repository separates the FameFlow work into three areas:

- `site/` - desktop website/workspace version.
- `app/` - mobile app prototype version.
- `design/` - SVG component kit, annotated spacing specs, export scripts, and presentation assets.

Open `index.html` for a simple entry page, or open each section directly:

- `site/index.html`
- `app/index.html`
- `design/outputs/svg-components/00-fameflow-component-kit.svg`
- `design/outputs/svg-components/00-fameflow-component-kit-annotated.svg`
